# ⚡ Lightzor

**Sitio web oficial para Lightzor – Electricistas en Santiago, RD.**

Este proyecto es una landing page desarrollada en HTML y CSS con el objetivo de captar clientes potenciales (leads) para servicios eléctricos residenciales e industriales, ofrecidos por técnicos locales de confianza.

---

## 🛠 Servicios que promociona Lightzor

- Instalación y mantenimiento de inversores
- Calentadores eléctricos
- Aires acondicionados
- Instalaciones eléctricas residenciales
- Proyectos eléctricos industriales

---

## 🚀 Objetivo

Facilitar la solicitud de servicios eléctricos a través de un sitio web ligero, rápido, accesible desde dispositivos móviles y conectado directamente con WhatsApp.

---

## 📲 Tecnologías usadas

- HTML5
- CSS3
- GitHub Pages (para despliegue)
- WhatsApp Click-to-Chat (integración de contacto)
